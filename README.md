In questo progetto ho sperimentato diverse tecniche avanzate di HTML e CSS, con un focus particolare sulle animazioni e sull'uso dei CSS Filters, per creare un effetto visivo dinamico e moderno.
Durante lo sviluppo ho approfondito diversi concetti tra cui:

🔹 CSS Animations e Transitions

Ho imparato a gestire:

Animazioni fluide tramite @keyframes
Timing functions per controllare la velocità delle animazioni
Transizioni applicate a hover ed eventi interattivi

🔹 CSS Filters

Ho esplorato i filtri CSS come:

blur()
brightness()
contrast()
grayscale()
e come combinarli per ottenere effetti estetici più complessi e accattivanti.

🔹 Modern Web UI

Il progetto mi ha aiutato a:

Migliorare la comprensione della costruzione di interfacce animate
Curare i piccoli dettagli visivi per rendere l'esperienza più interattiva
Sperimentare layout semplici ma eleganti
